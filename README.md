# STUDENT-GRADE-CALCULATOR
Python application to calculate student total marks, percentage, grade, and pass/fail status using loops and functions

This is a Python-based mini project developed to calculate:

- Total Marks
- Percentage
- Grade (A+, A, B, C, D, Fail)
- Pass/Fail Status

## Features
- Uses functions for modular programming
- Uses loops to take multiple subject inputs
- Includes input validation (0-100 range)
- Handles invalid inputs using exception handling

- Python
